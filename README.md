Educational Institute Network Design & Implementation
Overview

This project focuses on designing and implementing a Local Area Network (LAN) for an educational institute with multiple user groups, including students, teachers, administrative staff, and management. The objective was to create an efficient, reliable, and scalable network infrastructure within a single-floor building.

Objectives:

Design a network that supports 25+ users

Ensure efficient communication and resource sharing

Optimize performance, cost, and scalability

Implement and test network connectivity

Users & Resources:

20 Students

2 Teachers

2 Administrative Staff

2 Managers

1 Network Administrator


Devices:

10 Student Computers

1 Staff Computer

1 Admin Computer

1 Printer

Network Design

Topology Used: Star Topology

Network Type: Local Area Network (LAN)

Protocols: TCP/IP, HTTP, FTP, DNS, DHCP

The star topology was selected for its reliability, ease of troubleshooting, and scalability.

Devices & Components:

Router

Switch(es)

Server

Network Interface Cards (NICs)

Ethernet Cables

Devices were selected based on cost-effectiveness, performance, and ease of maintenance.

Implementation:

Designed network layout and blueprint

Assigned IP addressing scheme

Configured network devices

Connected all systems within a single-floor environment
 
Testing & Verification:

The network was tested using:

Ping (connectivity testing)

Traceroute (path analysis)

SSH (secure remote access)

All devices were successfully connected and verified.

Maintenance Plan:

Regular system monitoring

Device performance checks

Software and security updates

Troubleshooting and fault management.

Improvements & Recommendations:

Implement VLANs for better segmentation

Enhance security with firewalls and access control

Upgrade hardware for scalability

Introduce network monitoring tools
